New machine — first time
  1. Create a GitHub account if you don't have one — github.com/signup
  2. Send your GitHub username to Aritra. He grants repo access and you'll get an email invite — click Accept (the install can't clone until you do).
  3. Aritra sends you bootstrap.sh and shared.env — save both to your Downloads folder.
  4. Open Terminal (⌘-Space, type "Terminal") and paste:
bash ~/Downloads/bootstrap.sh --env-file ~/Downloads/shared.env
Already installed

Pull the latest and update — takes a few seconds:

cd ~/gtm-auto-dialer && git pull && bash update.sh
